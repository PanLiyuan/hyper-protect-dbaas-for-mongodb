# Datenbanken mithilfe des {{site.data.keyword.IBM_notm}} Support wiederherstellen
{: #restore_mongodb_databases}

Der {{site.data.keyword.cloud}}{{site.data.keyword.ihsdbaas_mongodb_full}}-Service löst einmal alle 24 Stunden automatisch eine Sicherung Ihrer gesamten Datenbank aus. Diese verschlüsselten Sicherungen sind für die letzten 7 Tage verfügbar.
{: shortdesc}

Wenn Sie eine Sicherung in eine vorhandenen Serviceinstanz wiederherstellen möchten, öffnen Sie eine Support-Anforderung:
1. Wählen Sie in der oberen Aktionsleiste **Support > Fälle verwalten > Neuen Fall erstellen** aus oder rufen Sie die [Supportseite](https://cloud.ibm.com/unifiedsupport/cases/manage){: external} direkt auf und klicken Sie auf **Neuen Fall erstellen**.
2. Wählen Sie die Art des Supports aus: **Technisch**
3. Wählen Sie die Kategorie **Datenbanken** aus.
4. Wählen Sie das Angebot **Name der Serviceinstanz oder des Datenbankclusters** aus.
5. Geben Sie im Betrefffeld zumindest "Wiederherstellen" ein.
6. Geben Sie im Beschreibungsfeld ein Datum innerhalb der letzten 7 Tage ein, das als Datum der Sicherung verwendet werden soll, von der das {{site.data.keyword.IBM_notm}} Team die Wiederherstellung durchführt.  
7. Aktivieren Sie das Feld, das angibt, dass Sie per E-Mail über Updates benachrichtigt werden.
8. Klicken Sie auf die Schaltfläche **Übergeben**.

Wenn die Wiederherstellung erfolgreich abgeschlossen wurde, werden Sie entsprechend vom {{site.data.keyword.IBM_notm}} Support-Team informiert. Alternativ können Sie den Status auf der Registerkarte **Support > Fälle verwalten** überprüfen. Weitere Informationen finden Sie im [Support Center](https://cloud.ibm.com/unifiedsupport/supportcenter){: external}.
